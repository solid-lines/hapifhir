# HAPIFHIR-docker-compose

Run: ./install.sh \<HOSTNAME\>

1. Update server
2. Add the given \<HOSTNAME\> to the configuration files
3. Build and create docker containers
4. Update Nginx configuration files
  
Go to https://\<HOSTNAME\>
